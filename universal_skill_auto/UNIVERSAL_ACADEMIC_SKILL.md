# 📘 通用学科学习 Skill / Universal Academic Skill

> 一个结构化、实践性强的学科学习框架，用于系统掌握任意领域的核心知识、思维模型、争议分析与理解检验。

---

## 🧭 一、Skill 概览

**用途 / Purpose**  
本 Skill 是一个可复用的学习体系模板，用于帮助学习者：
1. **全面理解**某个学科体系；
2. 收集与整理权威教材与专家资源；
3. 深度分析前沿论文与方法论；
4. 提炼思维模型并跨学科连接；
5. 建构批判性思维与争议分析；
6. 设计检验理解的习题与活动；
7. 持续反思与优化知识结构。

**适用对象 / Audience**  
- 自主学习者  
- 研究者与研究生  
- 企业学习团队  
- 学术顾问与导师

**使用前提 / Requirements**  
- 中英双语阅读能力  
- 基础学术阅读能力  
- Python 环境（用于自动化脚本辅助）

### 🌐 跨学科思考 Interdisciplinary Thinking

**定义 / Definition**  
跨学科思考是将多个领域的概念、方法与逻辑融合，用以更全面分析复杂问题的一种方法。它强调知识之间的联系，而不是单一视角的孤立理解。:contentReference[oaicite:1]{index=1}

**为什么重要 / Why It Matters**  
跨学科思考能够帮助学习者：
- 从不同领域获得启发性解释视角；
- 构建完整认知路径与概念网络；
- 在复杂问题中统一多个知识工具。

**在 Skill 中如何实践**  
每次阅读/思考时记录：
- 用到哪些其他学科的概念；
- 不同学科如何解释或补充当前主题；
- 是否能用另一个学科方法重新论证你的结论。

## 🧠 七、苏格拉底式提问 Socratic Questioning

**定义 / Definition**  
苏格拉底式提问是一种有结构的、系统性的提问方法，旨在通过提问揭示概念背后的假设、推理逻辑及其影响。它帮助学习者辨识已知与未知、深化理解并提升批判性思维。:contentReference[oaicite:2]{index=2}

**为什么重要 / Why It Matters**  
- 它能帮助你**区分事实、假设与推理结构**；  
- 通过质疑揭示隐藏前提与逻辑漏洞；  
- 促进学习者从被动接收转向主动思考。:contentReference[oaicite:3]{index=3}

### 🧠 经典提问类别

1. **澄清与解释类**  
   - “这个概念具体指的是什么？”  
   - “为何要这样定义？”  

2. **假设与证据类**  
   - “你的结论基于哪些假设？”  
   - “有哪些证据支持 / 反对它？”

3. **视角与替代解释**  
   - “有没有另一种可能的解释？”  
   - “如果用不同方法处理，会得出什么结果？”

4. **后果与影响**  
   - “这一结论导致哪些潜在后果？”  
   - “改变条件将如何影响结论？”:contentReference[oaicite:4]{index=4}

上述方法不仅用于课堂教学，在论文阅读、模型推导以及问题评估中都极具价值。

## 🛠 十一、自动化辅助脚本及使用说明

本仓库自动化脚本如下：

- `fetch_books.py` — 自动抓取教材/书目列表  
- `auto_summary.py` — 自动生成文章/论文摘要  
- `fetch_openalex.py` — 论文元数据查询脚本  
- `generate_quiz.py` — 自动生成理解检验题模板  
- `build_docs.sh` — 自动导出 Markdown 为 PDF / Word  

**基本用法示例：**  

pip install -r requirements.txt
python fetch_books.py
python auto_summary.py
python fetch_openalex.py
python generate_quiz.py
bash build_docs.sh
